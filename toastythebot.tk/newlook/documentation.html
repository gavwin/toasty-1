Documentation

;help - Sends link to list of commands and the Toasty official discord server for further help

;uwotm8 - Sends u wot m8 picture

;id - Sends your discord ID

;lenny - Sends the lenny face

 ;userinfo [@user] - Sends the info about the mentioned user

;avatar [@user] - Sends the avatar of the mentioned user

;stats - Sends the uptime, servers, channels and users of the bot

;invite - Sends OAuth link to invite Toasty to your server

;server - Tells you what server you are on

;uptime - Sends client uptime

;readytime - Sends time client emitted 'ready' (Time in unix timestamp)

;toast - Sends an img of toast

;say message - Says message

;serverinfo - Gives a list of info on the server including ID, date created, owner and more

;antiad enable - Enables anti advertisement feature

;antiad disable - Disables anti advertisement feature

;oauth - Sends OAuth2 / Invite link

;hq - Sends invite link to Toasty HQ server
